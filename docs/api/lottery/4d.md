# Lottery (4D) API

Documentation for lottery and 4D betting endpoints.

## Endpoints

### Get Latest Draw

```http
GET /api/lottery/4d/latest
```

**Description:** Get the latest 4D lottery draw results.

**Response:**

```json
{
  "status": "success",
  "draw_date": "2025-11-24",
  "draw_number": "1234",
  "results": {
    "first_prize": "1234",
    "second_prize": "5678",
    "third_prize": "9012"
  }
}
```

---

### Place Bet

```http
POST /api/lottery/4d/bet
```

**Description:** Place a 4D lottery bet.

**Request Body:**

```json
{
  "numbers": ["1234", "5678"],
  "bet_type": "big",
  "amount": 10.00,
  "draw_date": "2025-11-25"
}
```

**Response:**

```json
{
  "status": "success",
  "bet_id": "bet_123456",
  "total_amount": 20.00,
  "message": "Bet placed successfully"
}
```

---

### Get Bet History

```http
GET /api/lottery/4d/history
```

**Description:** Get user's 4D betting history.

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| limit | integer | Number of records to return |
| offset | integer | Pagination offset |

**Response:**

```json
{
  "status": "success",
  "bets": [
    {
      "bet_id": "bet_123456",
      "numbers": ["1234"],
      "amount": 10.00,
      "draw_date": "2025-11-25",
      "status": "pending"
    }
  ],
  "total": 10
}
```

---

### Check Winning

```http
POST /api/lottery/4d/check-winning
```

**Description:** Check if a bet is a winning bet.

**Request Body:**

```json
{
  "bet_id": "bet_123456"
}
```

**Response:**

```json
{
  "status": "success",
  "is_winner": true,
  "winning_amount": 5000.00,
  "prize_category": "first"
}
```
